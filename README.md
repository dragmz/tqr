# tqr

QR codes in terminal

### Usage

```go
package main

import (
	"fmt"

	"github.com/dragmz/tqr"
)

func main() {
	qr := tqr.New("https://github.com/dragmz/tqr")
	fmt.Println(qr)
}
```

`> go run examples/e1/main.go`
